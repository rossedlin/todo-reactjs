# Todo ReactJS

---

## Requirements

- node v18.3.0
- npm v8.19.1
- yarn v1.22.17
- firebase v11.8.1

## Setup

- `yarn create react-app todo-reactjs`
- `yarn add firebase react-icons`
- `yarn start`
- Navigate `https://localhost:3000/` and you'll see the spinning ReactJS Logo.
